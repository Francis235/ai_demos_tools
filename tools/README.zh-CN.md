## 📁 工具结构规范

每个工具应该遵循以下结构：

```
tool_name/
├── README.md           # 工具说明文档
├── requirements.txt    # Python依赖包（如果适用）
├── src/               # 源代码
│   ├── main.py        # 主程序入口
│   └── utils/         # 工具函数
├── config/            # 配置文件
├── examples/          # 使用示例
├── tests/             # 测试文件
└── docs/              # 详细文档（如果需要）
```

## 📖 工具文档模板

每个工具的README.md应该包含：

1. **工具名称和功能描述**
2. **主要特性**
3. **安装要求**
4. **使用方法**
5. **配置选项**
6. **使用示例**

## 🏷️ 工具标签

请在工具README中使用以下标签：

- `[命令行]` - 命令行工具
